# team13
## ゲームの目的
* 部屋の中に隠されたアイテムや仕掛けを見つけ出し、すべての謎を解き明かして部屋からの脱出を目指すゲームです。全ステージのクリアを目標に進行します。

## 基本操作方法
* 本作は、主にマウス操作のみでプレイすることができます。

* 探索する
- 気になる場所や怪しいオブジェクトをマウスでクリックします。

* 仕掛けを解く
- パズルやボタンなどが設置された場所をクリックすると、入力画面や謎解きギミックが起動します。画面の指示やヒントをもとに正しい操作を行います。

* ステージのやり直し（Rキー）
- ゲームプレイ中に R キーを押すと、現在のステージを最初からやり直すことができます。

* ステージ選択へ戻る（Bキー）
- ゲームプレイ中に B キーを押すと、ステージ選択画面へ戻ることができます。

## 遊び方の流れ（ステップガイド）
* ステージ選択画面
- ゲームを起動するとステージ選択画面が表示されます。遊びたいステージのドアをクリックしてゲームを開始します。

* 部屋の調査と謎解き
- 画面内のオブジェクト（ドアやパズルなど）をクリックして仕掛けを調べます。各ステージに用意された謎（ノック回数、順番入力、色合わせなど）を解きます。

* ドアのロック解除
- ステージ内のすべてのパズルや仕掛けをクリアすると、出口のドアのロックが解除されます（「OPEN」表示に変わります）。

* ステージクリア
- ロックが解除されたドアをクリックするとステージクリアとなり、クリア画面が表示されます。次のステージへ進むか、ステージ選択画面に戻ることができます。

# 脱出ゲーム

## 概要

* 部屋に隠されたアイテムや仕掛けを利用し、謎を解いて脱出を目指すゲーム

* 主にマウス操作を用いてステージのなぞ解きを行っていく。

## ルール

- プレイヤーが部屋の中をクリックなどのマウス操作で探索していく。
- 特定の場所を調べると*アイテム入手*や*仕掛け*を解くことができる。
- 入手したアイテムを適切な場所で使用することで謎解きを進めていくことができる。
- 全ステージ(約10ステージ？)をクリアすることで**ゲームクリア**となる。
